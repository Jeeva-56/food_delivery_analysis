# Project Name
Online Food Delivery Analysis: Data-Driven Business Insights

# Project Overview
This project focuses on analyzing an online food delivery dataset using Python, Pandas and MySQL to extract meaningful insights.

# What This Project Does
•	Processes and cleans the data using Pandas.
•	Stores the data in a MySQL database.  
•	Analyzes earthquake data using SQL queries.  

# TECHNOLOGIES USED
•	Python 
• Pandas 
• Matplotlib and Seaborn
• SQLAlchemy
• MySQL
• Power BI
